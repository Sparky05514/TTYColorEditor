# TTY Color Editor

A simple TUI text editor for Linux Virtual Consoles (TTY).
Allows you to customize the 16 standard ANSI colors of your TTY session.

## Features
- **RGB Sliders**: Visually adjust colors.
- **Presets**: Includes Matrix, Dracula, Solarized, etc.
- **Persistence**: Install to `~/.bashrc` to make themes permanent.
- **Live Preview**: See changes instantly.

## Usage
Run in a TTY (Ctrl+Alt+F3):
```bash
python3 tty_color_editor.py
```
- **Arrows**: Move
- **Enter**: Edit Color
- **P**: Presets
- **I**: Install/Permanent
- **S**: Save Script
- **Q**: Quit
